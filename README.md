# 🌍 World Atlas

World Atlas is a modern and fully responsive React.js web application that allows users to explore detailed information about countries around the world.

## 🚀 Features

- 🔎 Search any country
- 🌎 View country region & sub-region
- 🏳️ Country flag display
- 👥 Population details
- 💰 Currency information
- 🗣️ Languages spoken
- 🌐 Top-level domain
- 🏷️ Native country name
- 📱 Fully responsive design

## 🛠️ Tech Stack

- React.js
- React Router
- REST Countries API
- CSS (Responsive Design)

## 📌 Purpose

This project helps users quickly access and explore global country data in a clean and user-friendly interface.

Live Link: https://worldatlas3.netlify.app
